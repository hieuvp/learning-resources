## macOS

### Screenshot

- **Entire Screen**: **Shift** (⇧) + **Command** (⌘) + **3**
- **Selection** to **Clipboard**: **Control** (⌃) + **Shift** (⇧) + **Command** (⌘) + **4**
	- Then **Command** (⌘) + **V** to paste


## Application

### Firefox
